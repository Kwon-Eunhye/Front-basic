Semantic VS Non-semantic

- Non-semantic: div (의미없이 나누는 방식)
- semantic(의미가 있는): div를 제외하고 모든 element
- 만일 element가 고유한 attribute를 가지고 있다면 semantic을 non-semantic으로 대체할 수 없다

대체할 수 있는 semantic (div로 대체 가능)

  • header
  • nav
  • main
  • section
  • article
  • footer

대체할 수 없는 semantic (고유 속성이 존재)

  • image
  • a
  • form
  • input
  • label
